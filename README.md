# The Prefix RE in Latin and Romance

Interaktive Forschungs-Karte zur Erfassung linguistischer Befunde fuer italienische Staedte.

## Enthalten

- Interaktive Italien-Karte mit Markeransicht
- Kategorien fuer `ar-`, `re-/ri-`, `beide`, `unklar` und `keine Daten`
- Formular zur Befunderfassung mit Quelle und Kommentar
- Lokale Speicherung im Browser
- Export nach `CSV` und `XLSX`

## Lokale Nutzung

Die Anwendung ist eine statische Web-App und kann direkt aus dem Projektordner oder ueber einen einfachen lokalen Webserver geoeffnet werden.

## Hinweis

Die aktuelle Version nutzt noch keine gemeinsame Online-Datenbank. Der Mehrbenutzerbetrieb ueber `Supabase` ist als naechster Schritt vorgesehen.
